# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ecotwin-ai.git
```

Go to project

```bash
cd ecotwin-ai
```

Install dependencies

```bash
npm install
```

Create `.env.local`

```env
MONGODB_URI=
NEXTAUTH_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GROQ_API_KEY=
GEMINI_API_KEY=
```

Run the project

```bash
npm run dev
```

# 💡 Why EcoTwin AI?

EcoTwin AI goes beyond a traditional carbon calculator by creating a personalized digital climate twin. Users can visualize future environmental impact, track sustainability progress, and receive AI-powered recommendations to reduce their carbon footprint.
